Hello! Thanks if you want to contribute to the project.

## How to contribute?

To contribute to the project, you must fulfill the following requirements:
- What problems did you fix?

- What file did you modify?

- What library did you use?

- What files did you add?

- Did you use another language?

If you used another programming language, please specify cpp, c, or py.

## Requirements

The requirements to contribute are:

- Python 3.13
- Git
- Requires the typing, os, pathlib, and platform libraries.

## Format

The code format is very important.

Messages must always include the message type and details: "[Info] hello world!".

The code must be statically typed if you used a dynamically typed language that allows static typing.

Please do not use underscores (_) in variables.

# Step by step

1. Fork the repository.

2. Create your own branch (Required).

3. Make your changes.

4. Commit.

5. Submit your pull request.
